Class: WebExperiment

WebExperiment()

Orchestrates an A/B test defined by child `web-experiment-variant` elements. Determines which variant to display based on server pre-selection, persisted choice (cookie or local storage), or weighted random selection. Manages showing the chosen variant and hiding others.
